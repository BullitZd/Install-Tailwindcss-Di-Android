## Buka Aplikasi Termux
- update & upgrade paket
```bash 
   pkg update && pkg upgrade
```
- install NodeJs
```bash 
   pkg install nodejs
```
- cek apakah NodeJs sudah terinstall
```bash
   node -v
```
- cek apakah npm sudah terinstall
```bash
   npm -v
```
- install git
```bash
   pkg install git
```

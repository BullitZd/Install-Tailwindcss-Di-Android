## Buka Aplikasi Termux
1. Update & upgrade paket
```bash 
   pkg update && pkg upgrade
```
2. Install NodeJs
```bash 
   pkg install nodejs
```
3. Cek apakah nodejs dan npm sudah terinstall
```bash
   node -v
```
```bash
   npm -v
```
4. Install git
```bash
   pkg install git
```
